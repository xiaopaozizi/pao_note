1.push() 方法会将一个元素添加到数组末尾
var arr1 = [1, 2, 3, 4];
arr1.push(5);
console.log(arr1); // [1, 2, 3, 4, 5]
2.也可以使用数组的 length 属性为数组添加元素
var arr2 = [1, 2, 3, 4];
arr2[arr2.length] = 5;
console.log(arr2); // [1, 2, 3, 4, 5]
3.unshift() 方法可以将元素添加在数组的开头
var arr3 = [1, 2, 3, 4];
arr3.unshift(0);
console.log(arr3); // [0, 1, 2, 3, 4]
